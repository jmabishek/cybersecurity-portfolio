DAY - 1 NOTES

ls - list (shows files and folders in current spot)
   Can also peek into other folders: "ls foldername"
   Flags are like attachments for commands - they
   change what the command does.

   ls -a = shows hidden stuff too
   "."  = where I am right now
   ".." = one step up
   ls -l = shows permissions and details
       Looks like "drwx------"
       First letter tells the type
       d = directory, - = regular file
   ls -la = both at once (this is the one I actually use)
   ls -R = walks through everything inside, all levels

pwd - present working directory
   Just tells me where I am. That's it.
   Doesn't take arguments. "pwd foldername" won't work.

cd - change directory (walking around)
   cd foldername = go in
   cd .. = go up one level
   cd ~ = go home
   cd / = go all the way to the root
   cd - = jump back to where I just came from

clear - wipes the screen clean. Like erasing a whiteboard.

history - shows what I typed before
   history | head -20 = just the first 20
   Cybersec angle: attackers wipe history to hide.
   Investigators dig through history to find clues.
